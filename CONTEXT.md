# Sawt Atlas Urgence — صوت الأطلس
Assistant vocal de pré-triage médical offline pour le Maroc rural.
Application Expo React Native TypeScript.
L'APP PARLE EN PREMIER EN DARIJA. Le patient répond vocalement.
Interface : zéro texte obligatoire — voix + icônes + couleurs.
Protocole médical : AVPU + ABC + 9 domaines médicaux.
3 niveaux : 🔴 URGENCE EXTRÊME · 🟡 URGENCE RELATIVE · 🟢 NON URGENT
